# FireComply — 2 Minute Demo Script

## 0:00–0:15 — What it is
"FireComply helps MSPs turn raw Sophos firewall exports into deterministic security findings, compliance-aligned analysis, and client-ready reports in under two minutes."

## 0:15–0:30 — Remove friction
"Judges don't need a real config to test it, so I'll use the built-in demo mode."

- Click **Try Demo Config**

## 0:30–0:50 — Trust first
"Before any AI is involved, FireComply extracts and structures the firewall export client-side, shows parsing coverage, and computes deterministic findings. The config never leaves the browser in raw form."

- Point to **Extraction Summary**
- Point to trust/value strip

## 0:50–1:15 — Hero outcome
"This hero panel gives the answer immediately: overall posture, critical/high issues, compliance readiness, report readiness, and the top actions that would improve the score."

- Point to **Hero Outcome Panel**
- Call out **manual review time vs FireComply time**
- Point to **Top Actions to Improve Score**

## 1:15–1:35 — Deterministic analysis
"The Security Posture Scorecard is deterministic, not AI-generated. Each category is based on extracted configuration evidence."

- Point to **Security Posture Scorecard**
- Expand **Top 5 Critical Actions**
- Call out **Evidence Source** / **Extracted fact**

## 1:35–1:50 — Decision support
"This isn't just analysis — it's decision support. The Remediation Impact Simulator shows how the score, grade, and coverage improve if we apply recommended fixes."

- Open or point to **Remediation Impact Simulator**

## 1:50–2:00 — AI + output
"Once the deterministic layer has established the facts, FireComply uses AI to generate professional technical, executive, and compliance-ready reports, which can be exported immediately."

- Generate/open report
- Mention PDF / Word / PPTX / ZIP export

## Closing line
"So the value is: raw Sophos export in, trusted findings and executive-ready deliverables out — in minutes, not hours."
